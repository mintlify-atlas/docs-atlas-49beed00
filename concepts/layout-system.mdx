terminui provides a powerful **constraint-based layout system** for dividing rectangular areas into smaller sections.

## Core Concept

The layout system uses **constraints** to split a rectangular area into multiple sub-areas, either vertically (rows) or horizontally (columns).

```typescript
const layout = createLayout([
  lengthConstraint(3),       // exactly 3 rows
  percentageConstraint(50),  // 50% of remaining space
  fillConstraint(1),         // fill the rest
]);

const area = createRect(0, 0, 80, 24);
const [header, body, footer] = splitLayout(layout, area);
```

## Splitting Layouts

The `splitLayout` function takes a layout and an area, and returns an array of rectangles:

```typescript
const splitLayout = (layout: Layout, area: Rect): readonly Rect[]
```

From `~/workspace/source/src/core/layout.ts:325`

### Algorithm

1. **Apply margins** to get inner area
2. **Calculate spacing** between sections
3. **First pass**: Resolve all non-Fill constraints
4. **Second pass**: Distribute remaining space among Fill constraints
5. **Clamp sizes** to available space
6. **Build result rects** with proper offsets

```typescript
const splitLayout = (layout: Layout, area: Rect): readonly Rect[] => {
  if (layout.constraints.length === 0) {
    return [];
  }

  const inner = rectInner(area, layout.margin);
  const totalAvailable = layout.direction === 'horizontal' ? inner.width : inner.height;
  const count = layout.constraints.length;
  const gap = spacingAmount(layout.spacing);
  const totalGaps = count > 1 ? gap * (count - 1) : 0;
  const availableForConstraints = Math.max(0, totalAvailable - totalGaps);

  // First pass: resolve non-Fill constraints
  const sizes: number[] = [];
  let usedSpace = 0;
  let fillCount = 0;
  let fillWeightSum = 0;

  for (const constraint of layout.constraints) {
    if (constraint.type === 'fill') {
      sizes.push(0);
      fillCount++;
      fillWeightSum += Math.max(1, constraint.value);
    } else {
      const size = resolveSize(constraint, availableForConstraints);
      sizes.push(size);
      usedSpace += size;
    }
  }

  // Second pass: distribute remaining space among Fill constraints
  if (fillCount > 0) {
    const remaining = Math.max(0, availableForConstraints - usedSpace);
    let distributed = 0;

    for (let i = 0; i < count; i++) {
      const constraint = layout.constraints[i];
      if (constraint === undefined) {
        continue;
      }
      if (constraint.type !== 'fill') {
        continue;
      }

      const weight = Math.max(1, constraint.value);
      const share = Math.floor((remaining * weight) / fillWeightSum);
      sizes[i] = share;
      distributed += share;
    }

    // Give leftover pixels to the last Fill constraint
    const leftover = remaining - distributed;
    if (leftover > 0) {
      for (let i = count - 1; i >= 0; i--) {
        const constraint = layout.constraints[i];
        if (constraint === undefined) {
          continue;
        }
        if (constraint.type !== 'fill') {
          continue;
        }
        sizes[i] = (sizes[i] ?? 0) + leftover;
        break;
      }
    }
  }

  // Clamp sizes so total does not exceed available space
  let totalUsed = 0;
  for (let i = 0; i < count; i++) {
    const size = sizes[i] ?? 0;
    const clamped = Math.max(0, Math.min(size, availableForConstraints - totalUsed));
    sizes[i] = clamped;
    totalUsed += clamped;
  }

  // Build result rects
  const results: Rect[] = [];
  let offset = 0;

  for (let i = 0; i < count; i++) {
    const size = sizes[i] ?? 0;

    if (layout.direction === 'horizontal') {
      results.push(createRect(inner.x + offset, inner.y, size, inner.height));
    } else {
      results.push(createRect(inner.x, inner.y + offset, inner.width, size));
    }

    offset += size + gap;
  }

  return results;
};
```

## Examples

### Three-Row Layout

```typescript
const layout = createLayout([
  lengthConstraint(3),   // header: 3 rows
  fillConstraint(1),     // body: fill remaining
  lengthConstraint(1),   // footer: 1 row
]);

const area = createRect(0, 0, 80, 24);
const [header, body, footer] = splitLayout(layout, area);
```

Result:
- Header: 3 rows
- Body: 20 rows (24 - 3 - 1)
- Footer: 1 row

### Two-Column Layout

```typescript
const layout = createLayout(
  [
    percentageConstraint(30),  // sidebar: 30%
    fillConstraint(1),         // main: fill
  ],
  { direction: 'horizontal' },
);

const area = createRect(0, 0, 80, 24);
const [sidebar, main] = splitLayout(layout, area);
```

Result:
- Sidebar: 24 columns (30% of 80)
- Main: 56 columns (remaining)

### Weighted Fill

```typescript
const layout = createLayout([
  fillConstraint(1),  // 1/3 of space
  fillConstraint(2),  // 2/3 of space
]);

const area = createRect(0, 0, 60, 10);
const [left, right] = splitLayout(layout, area);
```

Result:
- Left: 20 columns (weight 1)
- Right: 40 columns (weight 2)

### Margins and Spacing

```typescript
import { createMargin } from 'terminui';

const layout = createLayout(
  [fillConstraint(1), fillConstraint(1)],
  {
    margin: createMargin(1, 2),  // 1 cell vertical, 2 cells horizontal
    spacing: { type: 'space', value: 1 },  // 1 cell gap between sections
  },
);
```

<Tip>
Constraints can be nested — split an area into sections, then split each section further with new layouts.
</Tip>
