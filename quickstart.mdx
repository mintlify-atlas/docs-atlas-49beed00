Build your first terminal UI with terminui in minutes. This guide covers both the functional API and the JSX API approaches.

## Functional API quickstart

<Steps>
  <Step title="Install terminui">
    Install terminui using your preferred package manager:
    
    <CodeGroup>
    ```bash npm
    npm install terminui
    ```
    ```bash yarn
    yarn add terminui
    ```
    ```bash pnpm
    pnpm add terminui
    ```
    </CodeGroup>
  </Step>

  <Step title="Create your first UI">
    Create a simple bordered paragraph widget:

    ```typescript
    import {
      createTestBackendState,
      createTestBackend,
      testBackendToString,
      createTerminal,
      terminalDraw,
      frameRenderWidget,
      blockBordered,
      createTitle,
      createParagraph,
      renderParagraph,
    } from 'terminui';

    // Set up a test backend
    const state = createTestBackendState(60, 10);
    const backend = createTestBackend(state);
    const terminal = createTerminal(backend);

    // Render a simple paragraph
    terminalDraw(terminal, (frame) => {
      const paragraph = createParagraph('Hello, terminui!', {
        block: blockBordered({ titles: [createTitle('Greeting')] }),
      });
      frameRenderWidget(frame, renderParagraph(paragraph), frame.area);
    });

    // Output the result
    console.log(testBackendToString(state));
    ```
  </Step>

  <Step title="See the output">
    Run your code to see the rendered output:

    ```
    ┌Greeting──────────────────────────────────────────────┐
    │Hello, terminui!                                      │
    │                                                      │
    │                                                      │
    │                                                      │
    │                                                      │
    │                                                      │
    │                                                      │
    │                                                      │
    └──────────────────────────────────────────────────────┘
    ```

    <Tip>
      The TestBackend is perfect for examples and testing. For production TUIs, you'll want to implement a backend that writes to `process.stdout` and enters the alternate screen.
    </Tip>
  </Step>
</Steps>

## JSX API quickstart

<Steps>
  <Step title="Configure TypeScript for JSX">
    Add JSX configuration to your `tsconfig.json`:

    ```json
    {
      "compilerOptions": {
        "jsx": "react-jsx",
        "jsxImportSource": "terminui"
      }
    }
    ```
  </Step>

  <Step title="Create a JSX component">
    Create a `.tsx` file with JSX components:

    ```tsx
    /** @jsxImportSource terminui */
    import {
      createTestBackendState,
      createTestBackend,
      createTerminal,
      testBackendToString,
      Color,
    } from 'terminui';
    import { terminalDrawJsx, Panel, Label } from 'terminui/jsx';

    const state = createTestBackendState(40, 6);
    const terminal = createTerminal(createTestBackend(state));

    terminalDrawJsx(
      terminal,
      <Panel title="Hello JSX" p={1}>
        <Label
          text="terminal UI, React-like syntax"
          fg={Color.Cyan}
          bold
          align="center"
        />
      </Panel>
    );

    console.log(testBackendToString(state));
    ```
  </Step>

  <Step title="Run with tsx">
    Execute your JSX file:

    ```bash
    npx tsx hello.tsx
    ```

    <Note>
      You can use `tsx`, `ts-node`, or compile with `tsc` and run with `node`.
    </Note>
  </Step>
</Steps>

## Build a layout with widgets

Create a more complex layout with multiple widgets:

```typescript
import {
  createTerminal,
  createTestBackend,
  createTestBackendState,
  terminalDraw,
  frameRenderWidget,
  frameRenderStatefulWidget,
  createLayout,
  splitLayout,
  lengthConstraint,
  fillConstraint,
  createList,
  createListState,
  renderStatefulList,
  gaugePercent,
  renderGauge,
  blockBordered,
  createTitle,
  testBackendToString,
} from 'terminui';

const state = createTestBackendState(60, 12);
const terminal = createTerminal(createTestBackend(state));

// Create stateful list
const list = createList(['Overview', 'Metrics', 'Logs'], {
  block: blockBordered({ titles: [createTitle('Menu')] }),
});
const listState = createListState(0); // First item selected

// Create gauge
const gauge = gaugePercent(42, {
  block: blockBordered({ titles: [createTitle('Load')] }),
});

terminalDraw(terminal, (frame) => {
  // Split the frame into two columns
  const layout = createLayout(
    [fillConstraint(1), fillConstraint(1)],
    { direction: 'horizontal', spacing: { type: 'space', n: 1 } }
  );
  const [leftArea, rightArea] = splitLayout(layout, frame.area);

  // Render list on the left
  frameRenderStatefulWidget(
    frame,
    renderStatefulList(list),
    leftArea,
    listState
  );

  // Render gauge on the right
  frameRenderWidget(frame, renderGauge(gauge), rightArea);
});

console.log(testBackendToString(state));
```
