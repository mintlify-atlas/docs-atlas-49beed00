title: Sparkline
description: Tiny inline data visualization widget
---

The **Sparkline** widget renders compact inline charts using unicode block characters. Perfect for displaying time-series data, metrics, and trends in minimal space.

## Visual Example

```
┌CPU───────────────────────────────────────────────────┐
│ ▁▄▂█▅▃▇▆                                             │
└──────────────────────────────────────────────────────┘
```

## Basic Usage

```typescript
import { createSparkline, renderSparkline, blockBordered, createTitle } from 'terminui';

const spark = createSparkline([1, 4, 2, 8, 5, 3, 7, 6], {
  block: blockBordered({ titles: [createTitle('CPU')] }),
  style: styleFg(createStyle(), Color.Cyan),
});

frameRenderWidget(frame, renderSparkline(spark), area);
```

## Configuration Options

### SparklineConfig

```typescript
interface SparklineConfig {
  readonly data: readonly number[];
  readonly max?: number;             // Override max value for scaling
  readonly block?: BlockConfig;
  readonly style: Style;
  readonly barSet: BarSet;           // Unicode bar characters
  readonly direction: 'leftToRight' | 'rightToLeft';
}
```

## Data Scaling

Sparklines automatically scale data to fit the available height:

```typescript
// Auto-scale to max value in data (8)
const auto = createSparkline([1, 4, 2, 8, 5, 3, 7, 6]);

// Force specific max value for scaling
const fixed = createSparkline([1, 4, 2, 8, 5, 3, 7, 6], {
  max: 10,  // Scale as if max is 10, not 8
});
```

**Auto-scaling**: The default behavior uses the maximum value in the data array.

**Fixed scaling**: Set `max` to scale multiple sparklines consistently or to show headroom.

## Direction

Control data rendering order:

```typescript
// Left to right (default)
const ltr = createSparkline(data, {
  direction: 'leftToRight',
});

// Right to left (recent data on right)
const rtl = createSparkline(data, {
  direction: 'rightToLeft',
});
```

### Use Cases

- **leftToRight**: Chronological data (oldest first)
- **rightToLeft**: Recent data emphasized (newest first)

## Bar Sets

Customize the unicode characters used:

```typescript
import { bar } from 'terminui';

// 9-level bars (default): ▁▂▃▄▅▆▇█
const spark1 = createSparkline(data, {
  barSet: bar.NINE_LEVELS,
});

// 3-level bars: ▁▄█
const spark2 = createSparkline(data, {
  barSet: bar.THREE_LEVELS,
});
```

### Available Bar Sets

```typescript
bar.NINE_LEVELS   // Empty, 1/8, 1/4, 3/8, 1/2, 5/8, 3/4, 7/8, Full
bar.THREE_LEVELS  // Empty, Half, Full
```

```typescript
import { styleFg, createStyle, Color } from 'terminui';

// Red for warnings

```

## Vertical Resolution

Sparklines use 8x vertical resolution for smooth rendering:

```typescript
// In a 3-row sparkline area:
// - 3 rows × 8 levels = 24 discrete heights
// - Data is scaled to 24 levels
// - Mapped to appropriate unicode characters

const data = [0, 5, 10, 15, 20];  // Max: 20
// Scaled to area height × 8
// Each value mapped to ▁▂▃▄▅▆▇█ characters
```

```typescript
const layout = createLayout([
  lengthConstraint(4),
  lengthConstraint(4),
  lengthConstraint(4),

// CPU sparkline

  frame,
```

## Time-Series Data

```typescript
// Rolling window of recent values
const recentMetrics: number[] = [];
const maxDataPoints = 50;

terminalLoop(
  terminal,
  (frame, tick) => {
    // Add new data point
    const newValue = Math.random() * 100;
    recentMetrics.push(newValue);

    // Keep only last N points
    if (recentMetrics.length > maxDataPoints) {
      recentMetrics.shift();
    }

    const spark = createSparkline(recentMetrics, {
      block: blockBordered({ titles: [createTitle('Live Metrics')] }),
      direction: 'rightToLeft',  // Recent data on right
      style: styleFg(createStyle(), Color.Cyan),
      max: 100,  // Fixed scale
    });

    frameRenderWidget(frame, renderSparkline(spark), frame.area);
  },
  { maxFrames: 200 }
);
```

## Complete Example

```typescript
import {
  createTestBackendState,
  createTestBackend,
  createTerminal,
  terminalDraw,
  frameRenderWidget,
  createLayout,
  lengthConstraint,
  fillConstraint,
  splitLayout,
  createSparkline,
  renderSparkline,
  blockBordered,
  createTitle,
  uniformPadding,
  styleFg,
  createStyle,
  Color,
} from 'terminui';

const state = createTestBackendState(70, 20);
const terminal = createTerminal(createTestBackend(state));

// Sample data
const cpuData = [12, 25, 18, 42, 35, 28, 50, 45, 38, 55, 48, 40];
const memoryData = [30, 32, 35, 38, 42, 45, 50, 48, 52, 55, 58, 60];
const diskData = [10, 15, 12, 18, 22, 20, 25, 28, 24, 30, 35, 32];
const networkData = [5, 8, 15, 12, 20, 18, 25, 22, 30, 28, 35, 40];

terminalDraw(terminal, (frame) => {
  const layout = createLayout([
    fillConstraint(1),   // CPU
    fillConstraint(1),   // Memory
    fillConstraint(1),   // Disk
    fillConstraint(1),   // Network
  ]);

  const [cpuArea, memArea, diskArea, netArea] = splitLayout(layout, frame.area);

  // CPU Sparkline
  frameRenderWidget(
    frame,
    renderSparkline(
      createSparkline(cpuData, {
        block: blockBordered({
          titles: [createTitle('CPU Usage')],
          borderType: 'rounded',
          padding: uniformPadding(1),
        }),
        style: styleFg(createStyle(), Color.Green),
        direction: 'leftToRight',
      })
    ),
    cpuArea
  );

  // Memory Sparkline
  frameRenderWidget(
    frame,
    renderSparkline(
      createSparkline(memoryData, {
        block: blockBordered({
          titles: [createTitle('Memory Usage')],
          borderType: 'rounded',
          padding: uniformPadding(1),
        }),
        style: styleFg(createStyle(), Color.Yellow),
        max: 100,  // Fixed scale
      })
    ),
    memArea
  );

  // Disk Sparkline
  frameRenderWidget(
    frame,
    renderSparkline(
      createSparkline(diskData, {
        block: blockBordered({
          titles: [createTitle('Disk I/O')],
          borderType: 'rounded',
          padding: uniformPadding(1),
        }),
        style: styleFg(createStyle(), Color.Cyan),
      })
    ),
    diskArea
  );

  // Network Sparkline
  frameRenderWidget(
    frame,
    renderSparkline(
      createSparkline(networkData, {
        block: blockBordered({
          titles: [createTitle('Network Traffic')],
          borderType: 'rounded',
          padding: uniformPadding(1),
        }),
        style: styleFg(createStyle(), Color.Magenta),
        direction: 'rightToLeft',
      })
    ),
    netArea
  );
});
```

<Note>
Sparklines are most effective in compact spaces. They use unicode block characters (▁▂▃▄▅▆▇█) for vertical resolution and automatically scale data to the available height.
</Note>
