## Overview

The Layout API provides functions for managing terminal layouts, defining size constraints, and working with rectangular areas.

## Types

### Position

```typescript
interface Position {
  readonly x: number;
  readonly y: number;
}
```

### Size

```typescript
interface Size {
  readonly width: number;
  readonly height: number;
}
```

### Rect

```typescript
interface Rect {
  readonly x: number;
  readonly y: number;
  readonly width: number;
  readonly height: number;
}
```

### Margin

```typescript
interface Margin {
  readonly top: number;
  readonly right: number;
  readonly bottom: number;
  readonly left: number;
}
```

### Layout

```typescript
interface Layout {
  readonly direction: Direction;
  readonly constraints: readonly Constraint[];
  readonly margin: Margin;
  readonly spacing: Spacing;
}
```

### Direction

```typescript
type Direction = 'horizontal' | 'vertical';
```

### Alignment

```typescript
type Alignment = 'left' | 'center' | 'right';
```

## Layout Functions

### createLayout

Creates a layout with the specified constraints.

```typescript
const createLayout = (
  constraints: readonly Constraint[],
  overrides?: {
    direction?: Direction;
    margin?: Margin;
    spacing?: Spacing;
  }
): Layout
```

<ParamField path="constraints" type="readonly Constraint[]" required>
  Array of constraints for layout segments
</ParamField>

<ParamField path="overrides" type="object">
  Optional layout configuration
  - `direction`: Layout direction ('horizontal' | 'vertical', default: 'vertical')
  - `margin`: Margin to apply (default: no margin)
  - `spacing`: Spacing between segments (default: no spacing)
</ParamField>

**Returns:** A new `Layout`

---

### splitLayout

Splits a rectangular area according to a layout.

```typescript
const splitLayout = (layout: Layout, area: Rect): readonly Rect[]
```

<ParamField path="layout" type="Layout" required>
  The layout definition
</ParamField>

<ParamField path="area" type="Rect" required>
  The area to split
</ParamField>

**Returns:** An array of `Rect` objects for each layout segment
