## Overview

The Text API provides functions for creating and manipulating styled text content using spans, lines, and text blocks.

## Types

### Span

```typescript
interface Span {
  readonly content: string;
  readonly style: Style;
}
```

### Line

```typescript
interface Line {
  readonly spans: readonly Span[];
  readonly style: Style;
  readonly alignment?: Alignment;
}
```

### Text

```typescript
interface Text {
  readonly lines: readonly Line[];
  readonly style: Style;
  readonly alignment: Alignment;
}
```

## Line Functions

### createLine

Creates a line from spans with optional style and alignment.

```typescript
const createLine = (
  spans: readonly Span[],
  overrides?: {
    readonly style?: Style;
    readonly alignment?: Alignment;
  }
): Line
```

<ParamField path="spans" type="readonly Span[]" required>
  Array of spans that make up the line
</ParamField>

<ParamField path="overrides" type="object">
  Optional configuration
  - `style`: Base style for the line
  - `alignment`: Text alignment ('left' | 'center' | 'right')
</ParamField>

**Returns:** A new `Line`
